Shirt product card

Trigger:

    Product Card:

    • Hover on
    • Hover off

    Heart Icon:

    • Hover On
    • Hover Off
    • Click

    Quick Add:

    • Hover On
    • Hover Off

    Size Labels

    • Hover On
    • Hover Off

    Size Button

    • Hover On
    • Hover Off

    Add Button:

    • Hover On
    • Hover Off
    • Click

Rules:

    Product Card:

      Hover on:
      • the image will fade to the second image over 0.5 seconds
      • the heart icon will reveal with a fade in over 0.5 seconds
      • the slide show arrows will reveal with a fade in over 0.5 seconds
      • the "Quick Add" buttons will reveal with a fade in over 0.5 seconds
      Hover off:
      • the second image will fade to the first image over 0.5 seconds
      • the heart icon will fade out over 0.5 seconds
      • the slide show arrows will fade out over 0.5 seconds
      • the "Quick Add" buttons will fade out 0.5 seconds

    Heart Icon:

      Hover on:
      • Heart icon gray border fades to a black 1px border over 0.5 seconds
      • Slideshow arrows fade out over 0.5 seconds on the product card
      • Cursor changes to finger
      Hover off:
      • Heart Icon black border fades to gray 1px border over 0.5 seconds
      • Slideshow arrows fade in over 0.5 seconds on the product card
      • Cursor changes to pointer
      Click:
      • If The heart grows to 125% size and fills with red
      • When clicked again, the heart shrinks back to 100% size and fills with white

    Quick Add:

      Hover on:
      • size button and add button box appear over .5 seconds
      Hover off:
      • size button and add button box fade out over .5 seconds

    Size Labels:

      Hover on:
      • size label gray border fades to a black 1px border over 0.5 seconds
      Hover off:
      • size label black border fades to a gray 1px border over 0.5 seconds

    Size Button:

      Hover on:
      • label box fades up from the "quick add" button over 0.5 seconds
      • 10% white overlay is added on top of the button over 0.5 seconds
      Hover off:
      • label box fades out from the "quick add" button over 0.5 second
      • 10% white overlay is removed from the top of the button over 0.5 seconds

    Add Button:

      Hover on:
      • 10% white overlay is added on top of the button over 0.5 seconds
      • Cursor changes to finger
      Hover off:
      • 10% white overlay is removed from the top of the button over 0.5 seconds
      • Cursor changes to pointer
      Click:
      • If the add button is clicked the background color of the button will fill to white and the text will switch to black over .5 sec
      • If the add button is clicked the background color of the button will reveal to black and the text will switch to white over .5 sec
